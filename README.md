### How to install

- Copy this command and run on your terminal.

```shell
git clone https://github.com/i386angel/kali-my-linux && cd kali-my-linux && sudo chmod +x kml.sh
```

<br>

___

<br>

### How to use

```shell
sudo ./kml.sh
```
